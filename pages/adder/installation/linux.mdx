# About Adder: Adder is a CLI tool for tailing the Cardano blockchain and emitting events for each block and transaction seen, based on user configurable filters. These events can be processed in other event-based toolchains to produce rich and reactive applications for Cardano.
 To install Adder for linux dowload the file from https://blinklabs.io/downloads
 Then open a  terminal go to your downloads folder
cd Downloads/ 
or the folder that you downloaded the file to then run this command to see the prompts to use the tools
./adder-v0.22.2-linux-amd64 --help 
to get desktop
